# Project Setup

## Backend

1. Initialize npm:
    ```bash
    npm init -y
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Start the backend server:
    ```bash
    npm start
    ```

## Frontend

1. Install dependencies:
    ```bash
    npm install
    ```

2. Run the frontend development server:
    ```bash
    npm run dev
    ```
